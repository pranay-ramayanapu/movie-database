## Setup Instructions

1. Clone the repository:
   ```sh
   git clone https://github.com/pranay-ramayanapu/movie-database.git
   cd movie-database
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
4. Open the app in your browser at `http://localhost:5173`
